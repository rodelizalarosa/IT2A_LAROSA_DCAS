# Dental Flow (Dental Clinic Appointment System)

Dental Flow is a desktop-based application designed to enhance the efficiency of dental clinics by managing patient information, appointment scheduling, and user roles. Its purpose is to streamline daily operations through role-based access control, providing specific functionalities for administrators, dentists, and patients. This system aims to improve workflow, minimize administrative errors, and elevate the quality of service in dental clinics by ensuring information is well-organized, secure, and easily accessible.

---

## ✨ Features

### 1. 🗂️ Manage Information

- 👥 Users Account management
- 🧑‍⚕️ Patient Information management
- 🏥 Staff Information management
- 🦷 Dentist Information management

### 2. 📅 Appointments

- 📆 Book an Appointment
- 🔄 Update Appointment Status
- 🧾 Generate Appointment Slip (for testing purposes)

### 3. 📋 Logs

- 🕒 Keep track of all system activities and events

### 4. ⚙️ Settings

- 👤 Profile Viewing and Update
- 🔐 Security features including Password Change

---

## 🛠️ Technology Stack

- **Frontend:** Java Swing (NetBeans)
- **Backend:** Java
- **Database:** MySQL (via XAMPP)
- **Platform:** Desktop Application

---

## 🚀 Installation

1. **📥 Download the Project:**

   [Clone or download the project files to your local machine.](https://github.com/rodelizalarosa/IT2A_LAROSA_DCAS.git)

2. **💻 Setup the Database:**

   - Open XAMPP and start MySQL.
   - Create a database for the application.
   - Import the provided SQL file located in the `database` folder into your MySQL database.

3. **▶️ Run the Application:**

   - Open the project in NetBeans.
   - Compile and run the application.
